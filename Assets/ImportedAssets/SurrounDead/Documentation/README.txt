For the demo scenes to work properly please make sure you follow the steps below:
1. Set Rendering Path to Deferred: Edit-> Project Settings -> Graphics -> Set Rendering Path to Deferred
2. Set Rendering Color SpaceSet: Edit-> Project Settings -> Player -> Set Rendering Color Space To Linear
3. For water to work properly you must import Standard Assets (Water) from documentation folder.
4. Download Post Processing Stack v2 from Window -> Package Manager -> Post Processing
5. If you get lots of errors after importing go to Window -> Package Manger-> Package Manager UI and update it.
6. Restart Unity to take effect.
